# Adams Docket URL section for any ADAMS search

Adams Docket URL section for any ADAMS search

## Usage

``` r
adams_docket_numbers(
  docket_numbers = NA,
  exclude = FALSE,
  closing_bracket = TRUE
)
```

## Source

<https://www.nrc.gov/site-help/developers/wba-api-developer-guide.pdf>

## Arguments

- docket_numbers:

  dbl/vector: Docket number (or numbers) to be searched on ADAMS

## Value

url segment for docket searches
