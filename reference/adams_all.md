# ADAMS all criteria searches URL section for any ADAMS search

ADAMS all criteria searches URL section for any ADAMS search

## Usage

``` r
adams_all(type_chr = NA, start_date = NA, end_date = NA)
```

## Source

<https://www.nrc.gov/site-help/developers/wba-api-developer-guide.pdf>

## Arguments

- type_chr:

  chr: Type of ADAMS document

- start_date:

  chr: The earliest date (ymd) search results should be returned

- end_date:

  chr: The latest date (ymd) search results should be returned

## Value

url segment for all criteria searches
