# ADAMS search posted date interval URL section for any ADAMS search

ADAMS search posted date interval URL section for any ADAMS search

## Usage

``` r
adams_start_or_end(date = Sys.Date(), end_lgl = TRUE)
```

## Source

<https://www.nrc.gov/site-help/developers/wba-api-developer-guide.pdf>

## Arguments

- end_lgl:

  Logical: TRUE formats a search before this date. FALSE formats a
  search after this date.

- date_chr:

  chr: The date (ymd)

## Value

url segment for publish date searches
