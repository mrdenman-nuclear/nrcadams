# Last URL section for any ADAMS search

Last URL section for any ADAMS search

## Usage

``` r
adams_search_tail(content_lgl = TRUE)
```

## Source

<https://www.nrc.gov/site-help/developers/wba-api-developer-guide.pdf>

## Arguments

- content_lgl:

  Default TRUE.

  - TRUE: Content search supports search keywords.

  - FALSE: Advanced search returns results for greater than 1,000
    entries.

## Value

Ending script for a ADAMS search URL
