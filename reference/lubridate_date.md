# Formats YMD date string into ADAMS search date using lubridate

Formats YMD date string into ADAMS search date using lubridate

## Usage

``` r
lubridate_date(date = Sys.Date())
```

## Source

<https://www.nrc.gov/site-help/developers/wba-api-developer-guide.pdf>

## Arguments

- date:

  chr: The date (ymd)

## Value

date for ADAMS search
