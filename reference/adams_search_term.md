# ADAMS search term URL section for any ADAMS search

ADAMS search term URL section for any ADAMS search

## Usage

``` r
adams_search_term(search_term)
```

## Source

<https://www.nrc.gov/site-help/developers/wba-api-developer-guide.pdf>

## Arguments

- search_term:

  chr: Any search term desired. Default is nothing (i.e., NA)

## Value

url segment for content searches
